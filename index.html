FALANDO UM POUQUINHO MAIS SOBRE MIM

Sou Luisa Leiria, pernambucana e estudante de Engenharia da Computação na Universidade Federal de Pernambuco. Durante a graduação, me encontrei no universo do Desenvolvimento Web e do UX Design — áreas que me permitem unir criatividade, pensamento analítico e empatia para resolver problemas reais.

Tenho experiência com tecnologias como HTML, CSS, JavaScript e React, além de familiaridade com ferramentas como Figma e Git. Gosto de transformar ideias em interfaces funcionais e agradáveis, sempre com foco na experiência do usuário.

Sou curiosa, dedicada e estou sempre em busca de aprender algo novo. Acredito que boas soluções digitais nascem da combinação entre código bem estruturado, design centrado no usuário e um olhar atento aos detalhes.

Que tal se conectar comigo em outras redes?

MEUS PRINCIPAIS PROJETOS

Sistema de Hotelaria - DestCination

Sistema de Hotelaria desenvolvido para a cadeira Engenharia de Software e Sistemas. Aplicação web completa para reserva de quartos de hotel, integrando conceitos de front-end, back-end, persistência de dados e boas práticas de engenharia de software. Tecnologias utilizadas: Frontend: React + TailwindCSS, Backend: Node.js + Express APIs REST, uso de sessionStorage para persistência temporária de dados Organização em sprints, versionamento com Git e colaboração contínua no GitHub
